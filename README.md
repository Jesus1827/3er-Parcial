**Descripcion:** Este es un proyecto diseñado para el desarrollo colaborativo de una aplicación web moderna utilizando tecnologías de react y vite. El objetivo principal de este repositorio es establecer una arquitectura limpia, modular y escalable que permita a un equipo de desarrolladores Frontend trabajar de forma simultánea sin conflictos.
